Welcome to TurnKey! You need to initialize this system first before you can use it. To do that you'll need to log into the root account via SSH. The turnkey-init initialization program will start automatically:

After initialization try reloading this page. This message will disappear and you'll be able to access all services on this system normally.

Can I log in without SSH?

No. Unfortunately not.

How do I log in via SSH?

If you are connecting from Linux or MacOS, please open your terminal app and type the command:

ssh example

If you are connecting from Windows, you will need to install an SSH client, such as PuTTY. Then connect to: IP ADDRESS OR FQDN

Note: you will need to have set a root password, or configured SSH keys before logging in
